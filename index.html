Dual Boot Clean Installation Guide

Here we are going to install Windows and Linux as dual boot.

Windows in HDD

Linux in SSD

HDD (300 GB)

SSD (128 GB)

This install Guide is For UEFI

First we will install Windows

Make a GPT bootable USB of windows, and start the setup.

At Install Now Wizard -> Launch CMD by pressing (Shift + F10)

Launch diskpart tool in CMD, and await for it to launch.
[x@y]:$
diskpart
Enlist all disks.
[x@y]:$
list disk
Select your HDD with its number.
[x@y]:$
select disk 0
wipe HDD.
[x@y]:$
clean
convert your HDD to GPT partition scheme.
[x@y]:$
convert gpt
create a new empty efi parition for persisting boot records.
[x@y]:$
create partition efi size=512
format this partition as fat32, and label it System.
[x@y]:$
format quick fs=fat32 label="System"
assign letter S to this partition.
[x@y]:$
assign letter="S"
create a new empty msr parition required by Windows.
[x@y]:$
create partition msr size=128
Select your SSD with its number.
[x@y]:$
select disk 1
wipe SSD.
[x@y]:$
clean
convert your SSD to GPT partition scheme.
[x@y]:$
convert gpt
exit diskpart tool.
[x@y]:$
exit
exit CMD.
[x@y]:$
exit

Proceed Installation Wizard -> Manual Installation

Create a new partition for C:\ in your HDD from Wizard

Select that C:\ partition and hit next to start installation.

After successfull installation of Windows

Now we will install Linux (MANJARO)

Make a bootable USB of Linux, and start the setup.

In its intallation wizard go for Manual Partitioning.

IN YOUR HDD, choose the efi partition with Fat32 file system.

Edit that partition, and don't format

just select the mount point /boot/efi

apply boot flag, if not already checked, and Hit OK.

Now we need to create one SWAP partition and one ext4 main partition for Linux in SSD

Create a new partition equivalent to the size of your RAM in MiB

Select File System: linuxswap

Format it

Must apply swap flag, and hit OK

Create a new partition ext4 of remaining size in SSD

Select File System: ext4

Select Mount Point: /

Format it

Must apply bios-grub flag, and hit OK

Now Select this ext4 partition and proceed to install.

For Username make sure that the name consists of single word. (No Spacing or Muliple Words) i.e dragon

Proceed with other info and install linux.

After successfull installation of Linux

Now we will update Linux (MANJARO)

Connect Internet, and Launch Terminal

remove pkg libcanberra-gstreamer if its present.
[x@y]:$
sudo pacman -R libcanberra-gstreamer
update the system.
[x@y]:$
sudo pacman -Syyu

After successfull update, restart

Install some useful utils from electrodragon installer, clone the installer
[x@y]:$
git clone 'https://github.com/electrodragon/installer.git'
change directory to installer
[x@y]:$
cd installer
Launch installer
[x@y]:$
python installer.py

Install mpv, youtube-dl, feh, xclip, scrcpy

Remove unwanted installer setup

Install some other utils
[x@y]:$
sudo pacman -S vim gparted gimp jdk11-openjdk jdk-openjdk python2
get ownership of /opt
[x@y]:$
sudo chown `whoami`:`whoami` /opt

Install Xampp, after successful installation, and its bin directory to your PATH

Install Composer

move composer.phar to directories visible to PATH, with the name of composer
[x@y]:$
sudo mv composer.phar /usr/bin/composer
Create SDKS directory for flutter and android-studio
[x@y]:$
sudo mkdir -p /sdks/Android/Sdk
Get Ownership of that directory
[x@y]:$
sudo chown -R `whoami`:`whoami` /sdks

For Flutter Place your sdk in /sdks
